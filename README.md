# Sistema Domótico IoT

Este proyecto consiste en la simulación de un sistema domótico utilizando Cisco Packet Tracer.

## Funcionalidades
- Automatización de iluminación
- Control de ventilación
- Monitoreo mediante sensores
- Control remoto desde smartphone

## Tecnologías
- Cisco Packet Tracer
- IoT
